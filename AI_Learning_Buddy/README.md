# 🎓 AI Learning Buddy

AI Learning Buddy is an AI-powered educational web application built using **Python**, **Streamlit**, and **Google Gemini AI**.

## Features

- 📘 Explain concepts in simple language
- 🌍 Provide real-life examples
- ❓ Generate multiple-choice quizzes
- 💬 Answer educational questions

## Technologies Used

- Python
- Streamlit
- Google Gemini API (Google GenAI SDK)

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/AbdulRaheema/AI-Learning-Buddy.git
   ```

2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Add your Gemini API key in Streamlit Secrets (or modify the app for local testing).

4. Run the application:
   ```bash
   streamlit run app.py
   ```
