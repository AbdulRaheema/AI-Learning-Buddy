# 🎓 AI Learning Buddy

AI Learning Buddy is an AI-powered educational web application developed using **Python**, **Streamlit**, and **Google Gemini AI**. It helps students understand concepts, generate quizzes, and receive AI-powered learning assistance through an interactive interface.

## Features

- 📘 Explains concepts in simple language
- 🌍 Provides real-world examples
- ❓ Generates multiple-choice quizzes
- 💬 Answers educational questions using AI

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- Google AI Studio
